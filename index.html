Thoughts on Progressive Enhancement

Progressive Enhancement is a web development philosphy that revolves around building a solid foundation of content and then enhancing that content progressively with new styles and features. Progressive Enhancement prioritizes the user experience by ensuring that the core of the content and functionality is accessible by all users. This is an inclusive system that aims to work with the current diverse set of devices and browsers that any user might use.

Progressive Enhancement is important to modern development becuase of its focus on accessiblity and future-profing. Progressive Enhancement prioritizes web content that is accessible to all users, regardless if that user is using an old device or not. By focusing on the foundation and functionality, this approach leaves no one left behind, locked out of the web for simply having an old device or browser. Next, the strong foundation that protects old users, also protects users from changes in the future.
